[Project Task Brief] GEMS/CemGEMS Thermodynamic Modeling:
Carbonation Equilibrium + Phase Diagrams + Reaction Pathways for a Composite Binder System

0. Basic Information
- Field/Topic: Resources & Environmental Engineering
- Deadline: Delivery before 2025-12-28
- System: Slurry system
Fly ash + Cement + Sodium silicate + Coal gangue + Water + CO₂
- Fixed conditions: T = 25°C; P_total = 1 atm; Closed system

1. Project Objectives (What you need to complete)
Under different mix proportions and CO₂ conditions, use GEMS (or an equivalent GEM-Selektor/CemGEMS workflow) to compute:
(1) Equilibrium stable phase assemblage and phase amounts (mainly solid phases)
(2) Key 2D phase diagrams (phase map + ternary diagram)
(3) Trend curves of key product contents versus variables
(4) Stepwise carbonation reaction pathways for 2–3 representative mix designs (phase transformation sequence / evolution curves)

2. Independent Variables and Values (Full-factorial scan)
There are 5 independent variables. The computation should be full-factorial:

(1) Binder-to-aggregate ratio
R = (Cement + Fly ash) / Coal gangue (mass ratio)
R ∈ {0.3, 0.6, 0.9, 1.2}

(2) Fly ash replacement ratio
f_FA = Fly ash / (Cement + Fly ash) (mass ratio)
f_FA ∈ {0, 0.1, 0.2, …, 1.0} (step 0.1, total 11 levels)

(3) CO₂ concentration in gas phase
yCO2 = volume fraction of CO₂ in the gas phase
yCO2 ∈ {0%, 15%, 20%, 25%, 30%, 35%, 40%} (7 levels)
Note: If using gas-phase partial pressure control, pCO2 can be set as pCO2 = yCO2 × 1 atm.

(4) Sodium silicate dosage in total slurry mass
w_SS = sodium silicate / total slurry mass (mass fraction)
w_SS ∈ {2%, 3%, 4%, 5%} (step 1%)

(5) Water-to-binder ratio
w/b = Water / (Cement + Fly ash) (mass ratio)
w/b ∈ {1.1, 1.4, 1.7, 2.0}

- Total equilibrium calculation points:
4 × 11 × 7 × 4 × 4 = 4928 cases
